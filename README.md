## 研究框架核心

1. **利润真实性**：大型云厂商的 AI 利润有多少是真实经营性现金流，多少来自循环融资和会计安排
2. **循环融资网络**：NVDA → CoreWeave / Oracle → OpenAI / Anthropic → Microsoft → NVDA 的封闭循环
3. **历史对照**：与 1999-2000 朗讯、思科电信泡沫的同构性分析
4. **三阶段崩盘剧本**：触发期（1-3 月）→ 重定价期（3-12 月）→ 出清期（12-36 月）
5. **AGI 立场**：基于现有架构和数据结构的科学约束，对"近年实现 AGI"持保留态度
